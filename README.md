#### 软件架构

基础框架：SpringBoot 2.7+

授权认证：sa-token（轻量级的Java权限认证框架）

高可用缓存：Redis

持久层：MyBatis Plus （支持动态数据源）

数据库连接池：Alibaba.druid

文件存储：minio

在线接口文档：SpringDoc

短信服务： 阿里云 ｜ 腾讯云

工具类： HuTool | IdGenerator

#### 开发部署

开发工具：idea
数据库版本： mysql8.0
redis版本： 5.0.14

- 启动redis

- 启动服务
  SimpleApplication

├─ simple-boot //父级工程  
│ │ ├─ simple-base //基础公共模块  
│ │ ├─ simple-center //系统管理业务功能  
│ │ ├─ simple-start //springboot启动模块
